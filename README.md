# Rounded Dark
Yet another (rounded) Dark Discord Theme.

![Theme Preview](https://adaelyn.needs.rest/r/Discord_tVlCsJ9S9w.png "Theme Preview")

- Fully Rounded UI design.
- Easily customizable values and colors.
